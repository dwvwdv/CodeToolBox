# 簡單操作、清晰易懂
## 快捷鍵 Hot-key
- D 刪除功能項目
- Esc 最小化圖標
- 雙擊最小化圖標打開應用
- 紅點拖拉窗口
- 紅點右鍵關閉程式
- 滑鼠中鍵新增自定義語言
